# 📖 자바스크립트 뼈까지 발라먹기

## 📌 스터디 개요
- **스터디 목표**: 바닐라 자바스크립트 뼈까지 발라먹기
- **스터디 방식**: 
  - 각자 책 또는 문서를 읽고 요약 및 정리
  - 스터디원과 개념 및 예상 면접 질문 공유

## 📂 폴더 구조
```
📁 js-deepdive-study
 ┣ 📂 week1  # 1주차 정리
 ┃ ┣ 📜 week1.md  # 개념 정리
 ┣ 📂 week2  # 2주차 정리
 ┃ ┣ 📜 summary.md
 ┗ 📜 README.md  # 리드미
```

## 📑 스터디 규칙
1. **미리 읽고 정리**: 주어진 범위를 미리 공부하고 핵심 내용을 요약합니다.
2. **질문과 토론**: 학습 범위 내 면접 질문 혹은 어려운 내용을 정리해서 공유합니다.
3. **기록 및 공유**: 학습 내용을 깃허브에 정리하여 공유합니다.
